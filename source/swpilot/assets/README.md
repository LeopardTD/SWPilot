# SWPilot Asset Store

This directory contains visual and media assets for the SWPilot project.

## Directory Structure

*   **`logos/`**: Application branding icons, system banners, and logo versions.
*   **`icons/`**: Standard interface vector SVG icons.
*   **`images/`**: Rasterized interface images (PNG, JPEG).
*   **`illustrations/`**: Narrative vector drawings and status illustrations.
*   **`animations/`**: Custom JSON animation assets.
*   **`lottie/`**: Lottie animation assets for interactive feedback states.
