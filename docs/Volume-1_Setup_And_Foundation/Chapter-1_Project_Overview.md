# Chapter 1 — Project Overview

## Tujuan

Dokumen ini menjelaskan gambaran umum proyek SWPilot, termasuk visi, tujuan, ruang lingkup, teknologi yang digunakan, dan target akhir pengembangan.

---

# Apa itu SWPilot?

SWPilot adalah aplikasi Android yang dirancang untuk membantu pengguna melakukan otomatisasi login harian pada game HoYoverse secara lebih praktis, aman, dan mudah digunakan.

Aplikasi ini memanfaatkan Accessibility Service Android untuk membantu menjalankan proses login secara otomatis berdasarkan konfigurasi yang telah ditentukan oleh pengguna.

---

# Visi Proyek

Membangun aplikasi otomatisasi login harian yang stabil, mudah digunakan, memiliki dokumentasi lengkap, serta dikembangkan menggunakan standar pengembangan perangkat lunak yang rapi.

---

# Tujuan Proyek

- Membuat aplikasi Android berbasis Flutter.
- Mengintegrasikan fitur native Android menggunakan Kotlin.
- Menyediakan sistem otomatisasi login yang modular.
- Menyusun dokumentasi proyek secara lengkap.
- Menggunakan AI sebagai pendamping pengembangan secara terstruktur.

---

# Ruang Lingkup

Pada versi awal, SWPilot akan difokuskan pada:

- Login harian akun HoYoverse.
- Pengelolaan akun.
- Pengaturan jadwal otomatisasi.
- Monitoring proses otomatisasi.
- Riwayat aktivitas.

Fitur lain akan ditambahkan pada versi berikutnya.
