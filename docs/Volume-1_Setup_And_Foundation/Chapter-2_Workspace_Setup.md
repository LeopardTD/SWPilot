# Chapter 2 — Workspace Setup

## Tujuan

Menyiapkan struktur workspace yang rapi sehingga seluruh aset, dokumentasi, source code, prompt, dan referensi tersimpan secara terorganisir.

---

# Struktur Folder

```text
SWPilot/
│
├── docs/
├── prompts/
├── assets/
├── references/
├── notes/
└── source/
```

---

# Fungsi Setiap Folder

## docs

Berisi seluruh dokumentasi proyek.

## prompts

Berisi prompt AI yang digunakan selama pengembangan.

## assets

Berisi logo, icon, gambar, ilustrasi, dan aset lainnya.

## references

Berisi referensi, artikel, dokumentasi resmi, serta catatan penelitian.

## notes

Berisi ide, bug, todo, meeting note, dan research.

## source

Berisi source code aplikasi.

---

# Workspace Rules

- Gunakan satu root folder untuk seluruh proyek.
- Hindari menyimpan file proyek di lokasi berbeda.
- Gunakan penamaan folder yang konsisten.

---

# Checklist

- [ ] Struktur folder telah dibuat.
- [ ] VS Code membuka folder root SWPilot.
