# Chapter 6 — Git Workflow

## Tujuan

Menjelaskan alur penggunaan Git selama pengembangan.

# Workflow

1. Pull repository.
2. Membuat branch baru.
3. Mengembangkan fitur.
4. Commit.
5. Push.
6. Pull Request.
